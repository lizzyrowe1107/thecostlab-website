Construction Software

Bluebeam

PDF Editor, used to view drawings.

Bluebeam is absolutely essential in the construction industry. If there is one software to master, it is Bluebeam. Project Engineers, Project Managers, Superintendents, and Estimators all use Bluebeam to review and markup drawings.

Excel

Spreadsheet software, used for tracking.

AutoCAD

Computer-Aided Design, used to create drawings.

This software is used to design and publish drawings.

Revit

Building Information Modeling, used to create a model.

This software is used to create a model and then drawings are generated by the model itself. Revit can incorporate CAD.
